## KeyDown 按住

`KeyPress` 键盘按住

- **示例**:

按键精灵8 及以上语法
```js
    此处粘贴代码

```

- 返回值: 

    无

- 函数原型:

    void KeyDown(`参数1`, `参数2`)

- 参数定义:

    - `参数1` **整形数**: 键盘虚拟码（8.X支持 按键字符 **字符型**）
    - `参数2` **整形数**: 次数
