# 加解密


##  Md5File 文件MD5加密

对路径文件进行MD5加密

- **示例**:

按键精灵8 及以上语法
```js
    //下面这句是对路径文件("C:log.txt")进行MD5加密FileMD5 = Plugin.Encrypt.Md5File("C:log.txt")
Msgbox FileMD5 

```

- 返回值: 

    字符串型，MD5值

- 函数原型:

    无

- 参数定义:

    - 参数1 字符串型，文件路径



按键精灵7 及以下语法:

```js
    - //下面这句是对路径文件("C:log.txt")进行MD5加密Plugin FileMD5 = Encrypt.Md5File("C:log.txt")
Msgbox FileMD5 
```
